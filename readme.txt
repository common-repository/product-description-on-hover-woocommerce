An add-on for WooCommerce to show Product Description in shop page on hover.

== Description ==

WooCommerce Product Description will show product description in tooltip when user hover on product image so that user don't have to go in to detail page to see desciption.

**Steps to make it work**

* Simply install the plugin.
* And It's Done. :)

**More features are available in Premium Plugin**

* Change position of tooltip
* Change color of tooltip
* Change theme of tooltip

We always welcome your suggestion and We are always there to support you.
